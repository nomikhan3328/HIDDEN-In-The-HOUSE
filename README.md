Create a modern, stylish, and fully responsive eCommerce website for a brand named "HIDDEN In The House". The design should feel minimal, mysterious, and premium (dark theme with elegant typography).

Tech Stack:

* HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla)
* No backend required (static site for GitHub Pages)

Pages Required:

1. Home (index.html)
2. Shop (products.html)
3. Product Details (product.html)
4. About (about.html)
5. Contact (contact.html)

Design Requirements:

* Dark theme (black, deep grey, gold/white accents)
* Clean and modern layout
* Smooth animations and hover effects
* Fully responsive (mobile + desktop)

Home Page:

* Hero section with brand name "HIDDEN In The House" and tagline:
  "Discover What Others Can't See"
* Call-to-action button: "Shop Now"
* Featured products section (4–6 items)
* Categories (e.g., Clothing, Accessories, Essentials)
* Testimonials section
* Footer with social links

Shop Page:

* Grid layout for products
* Product cards with image, name, price, and "View Details" button
* Filter or category buttons (optional)

Product Page:

* Large product image
* Title, price, description
* "Add to Cart" button (frontend only)
* Quantity selector

Cart System (JavaScript):

* Add to cart functionality using localStorage
* Cart icon with item count
* Simple cart page or dropdown

Contact Page:

* Form (Name, Email, Message)
* Basic validation using JavaScript

JavaScript Features:

* Mobile menu toggle
* Add to cart system (localStorage)
* Smooth scrolling
* Dark aesthetic animations

SEO & Optimization:

* Meta tags and titles for each page
* Fast loading and clean code
